# Local Skills Pack

This project vendors a small subset of skills from [pedronauck/skills](https://github.com/pedronauck/skills) for building a new Next.js institutional website.

## Added from pedronauck/skills

- `next-best-practices`
  - Next.js structure, App Router conventions, metadata, error handling, route handlers, and optimization rules.
- `react`
  - Component architecture, hooks, state boundaries, and TypeScript guidance.
- `tailwindcss`
  - Tailwind v4 patterns, token usage, and responsive layout conventions.
- `shadcn`
  - Component composition and customization patterns for shadcn/ui.
- `ui-craft`
  - UI quality guardrails for hierarchy, states, accessibility, microcopy, and design discipline.
- `a11y-testing`
  - Accessibility testing patterns with axe, Playwright, and jest-axe.

## Why this subset

These six cover the core work needed for an institutional site:

- framework decisions and App Router correctness
- component implementation
- styling and design-token discipline
- reusable UI composition
- page and section quality
- accessibility verification

I intentionally did not vendor the broader product, startup, or growth-oriented skills because they add noise for this repository's scope.
