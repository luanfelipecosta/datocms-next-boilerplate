# Website Skill Stack

This project uses `AGENTS.md` as the source of truth for:

- brand behavior
- brand tone
- factual constraints
- claim safety

Skills support execution. They do not override `AGENTS.md`.

## Installed core skills

- `brand-design-system`: deterministic Tailwind-based design-system page generation from Brand OS data
- `next-best-practices`: Next.js project structure, routing, rendering, metadata, and framework conventions
- `react-best-practices`: React implementation discipline for reusable sections and maintainable components
- `composition-patterns`: shared component APIs and composition patterns for scalable page sections
- `web-design-guidelines`: UI, UX, accessibility, and frontend quality review
- `site-architecture`: sitemap, navigation, content grouping, and page hierarchy
- `ui-craft`: visual direction and polished interface execution
- `brand-storytelling`: narrative structure and brand-aligned messaging support
- `copywriting`: section copy, CTA writing, and conversion-oriented phrasing
- `qa-report`: structured QA review before signoff

## Recommended usage order

1. `site-architecture`
2. `brand-storytelling`
3. `copywriting`
4. `ui-craft`
5. `next-best-practices`
6. `react-best-practices`
7. `composition-patterns`
8. `web-design-guidelines`
9. `qa-report`
